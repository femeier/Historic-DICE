DATA: The folder contains all input data,  the files for the historical spin-up and the calibration of the exogenous parameters and the other components of DICE.

OPTIMIZATION: Includes all AMPL files, there are two folders which differ in the SDR rate, the folder with "kappa" contains the optimization for calibrated values for \delta and \eta, th folder with "Drupp" contains the optimization for values of \delta and \eta from the expert assessment of Drupp et al. (2018).
In each folder is a model file (which is identical) and then different .run files for the different impact functions.

IW_calculations; the folder "Global" contains the MATLAB File to calculate the global IW assessment, in the Country-results we have calculated the IW country-specific results in Excel.

Figures: Here, the mathematica file summarizes the data and provides the result figures.
